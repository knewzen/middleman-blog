## Setup

**Installation**

```bash
$ yarn
```
or 
```bash
$ npm install
```

**Running**

```bash
$ yarn start
```
or
```bash
$ npm start
```
or 
```bash
$ middleman server
```

-- or --

```bash
$ yarn dev
```
or
```bash
$ npm run dev
```

open: [localhost:4567](http://localhost:4567)


### [middleman](https://middlemanapp.com/)#

```bash
$ middleman server
```

open: [localhost:4567](http://localhost:4567)
